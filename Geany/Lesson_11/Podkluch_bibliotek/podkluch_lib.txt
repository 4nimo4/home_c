
Подключение библиотек Linux

Для линковки с дополнительными библиотеками используются опции -l и -L.
Опция -l используется для указания имени библиотеки, с которой будет
производиться линковка.
При этом префикс lib отбрасывается.
Пример линковки с математической библиотекой libm:

gcc -lm -o program file1.o file2.o

Если библиотека расположена в нестандартном месте (стандартное место
зависит от платформы), то необходимо указать путь к ней опцией -L.
Пример линковки с указанием пути к библиотекам:

gcc -L/usr/local/lib -lm -o program file1.o file2.o
